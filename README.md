Web Form Validation

Purpose  
To create a web form that validates user input using JavaScript and prevents incorrect or incomplete data submission.

Features  
Input Fields: Textboxes for Name, Email, and Mobile Number.  
Validation: Checks for empty fields, valid email format, and 10-digit mobile number.  
User Feedback: Popup alerts guide users to correct errors before submission.

Technologies Used  
HTML5: Form structure.  
CSS3: Styling and layout.  
JavaScript: Input validation using conditional statements and regular expressions.

Setup Instructions  
Save index.html, style.css, and script.js in the same folder.  
Open index.html in any modern web browser.
